Project Description

This project facilitates the creation and management of user accounts, including activating and deactivating accounts. It consists of four main components: User Activation Form, User Deactivation Form, Password Change, and Branch Transfer. In the User Activation Form, users can create accounts for use in the portal and ERP system. After account creation, a manager can approve the account, and an SD (Service Desk) can finalize the process by clicking the finish button, enabling the user to access the selected systems. The User Deactivation Form operates similarly to the activation form, allowing managers and SD to deactivate accounts as needed. The Password Change component ensures that users can update their passwords, which will then be synchronized across all portal systems. The Branch Transfer Form allows users to transfer their account from one branch to another across all systems. This streamlined process ensures efficient account management and system access control.
